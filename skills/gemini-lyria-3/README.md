# 🎵 Gemini Lyria-3（AI 音樂生成）

> 給一段歌詞或描述，小龍蝦幫你用 AI 作一首歌 🦞🎶

## ✨ 這個技能可以做什麼？

- 🎵 文字生成音樂——描述你想要的風格或直接貼歌詞，AI 幫你作曲
- 📝 歌詞轉音軌——提供完整歌詞，Lyria-3 生成與歌詞語調相符的音樂
- 🎼 多段音軌輸出——一次生成多個音軌片段，自動存檔
- 📁 自動整理結果——音訊檔案存入指定目錄，並回報下載連結

## 📦 安裝方式

在 Telegram 對話中輸入：

```
/skills
```

輸入後會顯示可安裝的技能列表，選擇此技能即可完成安裝。

## ⚙️ 需要的設定

| 設定項目 | 說明 |
| --- | --- |
| `GEMINI_API_KEY` | Google Gemini 的 API 金鑰（**必填**） |

## 💬 提示詞範例

以下是你可以直接複製貼上給小龍蝦的提示詞：

```text
幫我根據這段歌詞生成一首歌：「夜深了，城市裡的燈光還亮著...」
用 Lyria-3 生成一段輕鬆的爵士樂
幫我作一首適合冥想的純音樂，安靜、空靈的風格
根據以下歌詞生成台語風格的音樂：...
```

## 📝 輸出範例

生成完成後，小龍蝦會回報：

```text
🎵 音樂生成完成！

- [track-0.mp3](https://github.com/your-repo/blob/issue-3/music-output/track-0.mp3?raw=true)
```

音軌會自動儲存到指定的輸出目錄中。

## ⚠️ 注意事項

- 預設使用的模型是 `lyria-3-pro-preview`
- 歌詞或提示詞必須事先寫入文字檔，再指定給腳本
- 若 API 回傳多段音訊，會依序儲存為 `track-0.mp3`、`track-1.mp3`…
- 需要有效的 Gemini API 金鑰（具有 Lyria 模型存取權限）才能使用
