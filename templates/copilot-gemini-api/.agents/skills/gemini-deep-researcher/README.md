# 🛠️ Deep Researcher（深度研究報告）

> 給小龍蝦一個主題，牠會幫你深度研究、整理資料，生出一份有引用來源的完整報告！

## ✨ 這個技能可以做什麼？

- 📚 針對任何主題產生一份**結構完整的深度研究報告**
- 📎 報告中的每個結論都會附上**引用來源**，方便查證
- 🇹🇼 輸出全程使用**繁體中文**，不用再自己翻譯
- 🏗️ 報告自動分成：摘要、背景、主要發現、分析、結論與建議、參考來源
- 🔬 底層使用 Gemini Deep Research 代理，會自動搜尋大量資料來源

## 📦 安裝方式

在 Telegram 對話中輸入：

```
/skills
```

輸入後會顯示可安裝的技能列表，選擇此技能即可完成安裝。

## ⚙️ 需要的設定

使用前需要設定以下 API Key：

| 名稱 | 說明 |
| --- | --- |
| `GEMINI_API_KEY` | Google Gemini API 金鑰 |

請到 [Google AI Studio](https://aistudio.google.com/) 自行申請。

## 💬 提示詞範例

以下是你可以直接複製貼上跟小龍蝦說的話：

```text
幫我研究「台灣半導體產業供應鏈」，產出一份完整的分析報告。
請深度調查 AI 晶片市場的現況與未來趨勢。
我想了解量子運算對現代密碼學的影響，幫我寫一份報告。
研究一下全球電動車市場的競爭格局，附上參考來源。
幫我比較三大雲端平台（AWS、GCP、Azure）的優缺點，寫成報告。
```

## 📝 輸出範例

小龍蝦會回傳一份像這樣的 Markdown 報告：

```
## 摘要
本報告探討台灣半導體產業在全球供應鏈中的關鍵地位……

## 背景
台灣半導體產業自 1980 年代起步，經過數十年發展……

## 主要發現
1. 台積電佔全球晶圓代工市場超過 50% 的份額……
2. 先進製程（3 奈米以下）集中在台灣生產……

## 分析
從地緣政治角度來看，台灣半導體產業面臨……

## 結論與建議
建議持續投資先進製程研發，同時分散生產基地……

## 參考來源
- [1] Bloomberg, "TSMC's Global Dominance...", 2024
- [2] 工業技術研究院, 《半導體產業年報》, 2024
```

## ⚠️ 注意事項

- ⏳ 研究需要 **2～8 分鐘**，請耐心等待！小龍蝦正在認真查資料
- 🇹🇼 不管你用什麼語言提問，報告**一律以繁體中文輸出**
- 📊 Gemini API 有使用量限制，如果頻繁使用可能會遇到配額上限
- 💡 主題描述越清楚具體，報告品質就越好
